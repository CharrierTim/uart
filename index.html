    UART PROJECT¶

    Documentation for the UART Project targeting the Zedboard device.

    Overview¶

    FPGA project for the Zedboard centered on a UART peripheral and an AXI-Lite register block to control IPs/peripherals (SPI, VGA, etc.)

    Control & status register (CSR) automation is done with SystemRDL and PeakRDL-regblock-vhdl.

    Most of the design is built from scratch, but some IPs are taken from the Open Logic FPGA Standard Library.

    Verification uses the VUnit framework with code coverage across open-source and commercial simulators:

    • nvc simulator
    • GHDL simulator
    • QuestaSim/ModelSim simulator

    Synthesis and implementation are done with Vivado.

    Clocking Configuration¶

    The FPGA uses a PLL (clk_wiz_0) to generate internal clocks from the input clock.

    Signal Name Type Frequency Description
    PLL Input
    clk_in1 Clock 100 MHz Input clock from PAD_I_CLK (Y9)
    PLL Outputs
    clk_out1 Clock 50 MHz Internal system clock (internal_clk)
    clk_out2 Clock 65 MHz VGA clock (vga_clk)
    locked Status - PLL lock indicator (pll_locked)
